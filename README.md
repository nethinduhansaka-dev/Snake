# Modern Snake Game 🐍

A modern implementation of the classic Snake game with enhanced graphics and features, built using vanilla JavaScript and HTML5 Canvas.

![Snake Game Demo](screenshots/demo.gif)

## 🎮 Features

- Smooth, responsive controls
- Modern gradient-based graphics
- Advanced scoring system with local high score tracking
- Progressive difficulty levels
- Customizable snake speed
- Pause/Resume functionality
- Mobile-friendly design
- Grid-based movement with visual guides
- Visual effects including:
  - Snake gradient coloring
  - Food particle glow effects
  - Snake head details
  - Smooth animations

## 🚀 Getting Started

### Prerequisites

- Any modern web browser (Chrome, Firefox, Safari, Edge)
- Local web server (optional, for development)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/snake-game.git
```

2. Open the `index.html` file in your web browser or serve it through a local web server.

### Playing the Game

- Use Arrow keys or WASD to control the snake
- Collect the red food particles to grow and score points
- Avoid hitting the walls or the snake's body
- Press Space or P to pause/resume the game
- Use the speed slider to adjust the snake's movement speed

## 🎯 Game Controls

- **Movement**: Arrow keys or WASD
- **Pause/Resume**: Space bar or P key
- **Speed Control**: Slider (5 levels from Very Slow to Very Fast)
- **Restart**: Button click after game over

## 🛠️ Technical Details

### Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- HTML5 Canvas API
- Local Storage API

### Code Structure

- `index.html`: Main game structure and layout
- `<style>`: CSS styling for the game interface
- `<script>`: Game logic and canvas rendering

### Key Components

1. Game State Management
   - Score tracking

   - High score persistence

2. Rendering Engine
   - Gradient-based graphics
   - Grid system
   - Snake and food rendering
   - Visual effects

3. Input Handling
   - Keyboard controls
   - Touch input support
   - Speed control system

## 🔄 Game Loop

The game operates on a main loop that:
1. Updates game state
2. Handles collisions
3. Processes input
4. Renders graphics
5. Adjusts difficulty
